  • 是的，娱乐媒体和海盗湾等软件的数字内容的在线索引是合法的，但是在没有首先获得版权所有者的权利的情况下，受版权保护的内容的分发是不合法的。
  • 由于The Pirate Bay只是一个庞大的所谓磁铁链接数据库，它允许在不需要连续可用的主机的情况下参考资源， 因此它实际上并没有违反任何版权法 。
